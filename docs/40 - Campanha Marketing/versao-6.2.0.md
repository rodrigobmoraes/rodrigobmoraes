# Versão 6.2.0

# Melhorias

## **BIP-2141 Permitir ao administrador do sistema de campanhas de MKT incluir NFs no sistema.**

Ajustado a permissão do usuário no envio de notas fiscais, caso seja um usuário administrador do módulo de campanha de marketing, é permitido o envio de notas fiscais.

![Docusaurus logo](/img/bip-2141-jira.png)

## **BIP-2203 Desenvolver as notificações de e-mail dos processos de workflow do fluxo da Campanha de Marketing.**

Realizado o desenvolvimento das notificações de e-mail dos processos de workflow do fluxo do sistema de campanha de marketing.

## **BIP-225 Ajustar apuração das campanhas de cooperantes para considerar NFs confirmadas.**

A rotina de apuração de cooperantes, foi devidamente ajustado para considerar NFs com status "Confirmada". Independente da sua espécie.
